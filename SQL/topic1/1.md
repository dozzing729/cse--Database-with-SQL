## SELECT

SELECT 문은 테이블에서 원하는 컬럼의 데이터만 선택해 출력한다

 
 - 산술식을 포함할 수 있다

  ```sql 
  SELECT ID, name, salary/2
  ```
  
 - 집계함수를 사용할 수 있다

 - AS를 이용하여 각 컬럼에 alias를 설정할 수 있다
  
  ```sql
  SELECT salary / 2 AS 'newSalary'
  ```
  
  - 중복되는 행 제거 or 포함: DISTINCT, ALL(default)

  ```sql
  SELECT DISTINCT salary
  ```
  
 
 
 ## WHERE
 
 WHERE 문을 이용하여 특정 조건식을 만족하는 행만 선택해 출력한다
 
 다음과 같은 방법을 여러가지 조건식을 나타낼 수 있다
 
 
 - 구간 설정
 
   - 'BETWEEN' a 'AND' b : a <= x <= b
   
   - 'NOT BETWEEN' a 'AND' b : x < a or b < x

 - 문자열 패턴 매칭 
 
   - (속성) 'LIKE' (표현식) : 해당 속성이 표현식을 만족하는지 확인

   자주 사용하는 표현식들 !

   - '%' : 임의의 길이, 임의의 문자를 갖는 문자열을 의미한다
   - '_' : 임의의 문자 "한개"를 갖는 문자열을 의미한다
   
   ex) 'c___@%' : c로 시작하고, 아무 글자 3개, @, 그 뒤에 임의의 문자열이 붙는 문자열을 의미한다
