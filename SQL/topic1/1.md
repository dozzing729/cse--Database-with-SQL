## SELECT

SELECT 문은 테이블에서 원하는 컬럼의 데이터만 선택해 출력한다

 
 - 산술식을 포함할 수 있다

  ```sql 
  SELECT ID, name, salary/2
  ```
  
 - 집계함수를 사용할 수 있다

 - AS를 이용하여 각 컬럼에 alias를 설정할 수 있다
  
  ```sql
  SELECT salary / 2 AS 'newSalary'
  ```
  
  - 중복되는 행 제거 or 포함: DISTINCT, ALL(default)

  ```sql
  SELECT DISTINCT salary
  ```
  
 
 
 ## WHERE
 
 WHERE 문을 이용하여 특정 조건식을 만족하는 행만 선택해 출력한다
 
 다음과 같은 방법을 여러가지 조건식을 나타낼 수 있다
 
 
 - 구간 설정: **BETWEEN AND**
 
   - BETWEEN a AND b : a <= x <= b
   
   - NOT BETWEEN a AND b : x < a or b < x

 - 원하는 값들의 속성만 조회: **IN**

   - (속성) IN (a1, a2, a3) : 속성의 값이 a1 또는 a2 또는 a3인 행만 출력한다. 

 - 문자열 패턴 매칭: **LIKE**
 
   - (속성) LIKE '(표현식)' : 해당 속성이 표현식을 만족하는지 확인

   자주 사용하는 표현식들 !

   - % : 임의의 길이, 임의의 문자를 갖는 문자열을 의미한다
   - _ : 임의의 문자 "한개"를 갖는 문자열을 의미한다
   - BINARY : 
   
   ex) 'c___@%' : c로 시작하고, 아무 글자 3개, @, 그 뒤에 임의의 문자열이 붙는 문자열을 의미한다
   
 - 여러개의 조건을 AND/OR로 연결할 수 있다



### 추가) Date type

yyyy-MM-DD 의 형태로 여러가지 유용한 함수를 알아놓자 


- 부등호 연산 가능 (무야호!)

- 연/월/일 추출: **YEAR(date a) / MONTH(date a) / DAYOFMONTH(date a)**

  ```sql
  WEHRE YEAR(birthday) = '1992'
  ```
 
- 날짜 간의 차이 : **DATEDIFF(date a, date b)**

- 오늘 날짜 : **CURDATE()**

- 특정 시간 기준으로 더하거나 빼기 : **DATE_ADD(date a, INTERVAL n 단위) / DATE_SUB(date a, INTERVAL n 단위)**

  - date a를 기준으로 n 단위 만큼을 더하거나 뺀 값을 date 타입으로 반환한다.

  ```sql
  WHERE MONTH(DATE_ADD(birthday, INTERVAL 10 DAY)) = 12
  ```
